Introduction

I recently shared tutorials on using Quarto, the next generation of RMarkdown, and Murtada Khalafall asked me to do a Quarto tutorial with R. So, this tutorial is the R version of my previous Quarto tutorial 3. yaml options are the same for both R and Python, except that the jupyter is set to Python (jupyter: python3) when using Python for data analysis.

Quarto

Quarto enables you to weave together content and executable code into a finished document. To learn more about Quarto see https://quarto.org.

Code 
from math import pi, e

print(f'pi, rounded to 3 decimal places, is {pi:.3f}.')
pi, rounded to 3 decimal places, is 3.142.
Code 
print(f'e, rounded to 3 decimal places, is {e:3f}.')
e, rounded to 3 decimal places, is 2.718282.
